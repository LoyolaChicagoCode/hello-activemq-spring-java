Thanks to Stefan for having provided this extremely helpful example on his blog:

http://stefdb.wordpress.com/2009/09/10/asynchronous-messaging-with-embedded-activemq-and-spring-jms/

Required Maven version:

  This project needs Maven 2.x. 

To install dependencies:

  1) First run 

    mvn install

  2) Then follow the resulting instructions.

  3) If you are studying this example in the context of a class,
     your instructor might have given you additional information
     on downloading the missing jars.

To run:

mvn exec:java -Dexec.mainClass=com.wordpress.stefdb.activemq.Main
